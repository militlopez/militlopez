# ¡Hola, soy Milagros! 🖐🏻
## Sobre mí
Mi nombre es Milagros López, soy desarrolladora Front End de Buenos Aires, Argentina, enfocado en crear interfaces limpias, rápidas y accesibles. Me encanta programar 💜.

Soy autodidacta y realicé cursos de HTML, CSS, JavaScript y Bootstrap. Me adapto con facilidad a nuevos lenguajes de programación, 
frameworks y librerías.

- ✍🏻 **Hobbies:** leer sobre nuevas tecnologias,  desarrollar pequeños proyectos de  programación para probar cosas nuevas, aprender inglés, organizar eventos variados (club de lectura, visualización y juegos varios)  y realizar actividad fisica.
- 🎯 **Objetivos:** mi objetivo a corto plazo es transformar esa curiosidad en mi primer trabajo como developer; a largo plazo, sumarme a comunidades y proyectos reales, no solo para aprender sino también para aportar conocimiento.
